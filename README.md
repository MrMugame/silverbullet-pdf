# Silverbullet PDF
This is an experimental new version of Silverbullet PDF, which is custom build using pdfjs and svelte. It's very similar and also uses code from [PDF++](https://github.com/RyotaUshio/obsidian-pdf-plus). Development is currently on hold as I lost motivation, don't know if I'll continue, so feel free to fork.

![screenshot](https://raw.githubusercontent.com/mrmugame/silverbullet-pdf/v2/docs/preview.png)

## Possible future features
- Embeds (currently not supported by SB)
- Remove svelte name mangling for custom styling
- Thumbnail viewer
- Different colors and link templates
